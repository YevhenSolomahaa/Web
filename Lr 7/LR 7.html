Тема: Практика роботи зі строковими даними в JavaScript

Завдання 1: Поточний час у форматі

Формат: 09:15:56, середа, 06 травня 2025 року

Пояснення коду Завдання 1
// Функція showCurrentTime()
1. new Date() - створює об'єкт дати з поточним моментом часу
2. getHours(), getMinutes(), getSeconds() - отримуємо години, хвилини, секунди
3. padStart(2, '0') - додає нуль спереду, якщо число менше 10 (наприклад, 9 → "09")
4. Масив days[] - містить назви днів тижня українською мовою
5. getDay() - повертає день тижня (0 = неділя, 1 = понеділок, ...)
6. getDate(), getMonth(), getFullYear() - отримуємо число, місяць, рік
7. getMonth() + 1 - місяці в JS від 0 до 11, тому додаємо 1
8. Виводимо рядок у шаблонному форматі з допомогою зворотних лапок `${}`
9. console.log() - виводить у консоль браузера (F12)
10. document.getElementById().innerHTML - виводить на сторінку

Завдання 2: Гра "Вгадай число"

Правила: Комп'ютер загадує число від 0 до 50. Ви маєте його вгадати.
Підказки: гаряче (різниця ≤5), тепло (різниця ≤15), холодно (різниця >15)
Кожна спроба записується в консоль із часом.

Пояснення коду Завдання 2
// Основна функція guessTheNumber()

1. Цикл while (playAgain) - дозволяє грати багато разів
playAgain = true → граємо, false → вихід

2. Генерація числа: Math.random() → число 0..0.999, множимо на 51 → 0..50.999, Math.floor() → ціле 0..50

3. Внутрішній цикл while (!guessed) - поки не вгадано

4. prompt() - показує модальне вікно з полем вводу
Якщо користувач натиснув Cancel → повертає null → виходимо з гри

5. Number(userInput) - перетворює рядок на число
isNaN() перевіряє, чи це число (NaN = Not a Number)

6. Логування спроби в консоль:
new Date() → отримуємо час спроби
Форматуємо дату: день.місяць.рік
Форматуємо час: години:хвилини:секунди
console.log() виводить у консоль браузера

7. Перевірка вгадування:
if (guess === secretNumber) → вгадав → alert з результатом
guessed = true → вихід із внутрішнього циклу

8. Підказки:
Math.abs() → абсолютне значення різниці
if (diff ≤ 5) → "гаряче"
else if (diff ≤ 15) → "тепло"
else → "холодно"

9. alert() - показує підказку

10. Після вгадування: confirm() → питає "Хочеш зіграти ще раз?"
true → playAgain = true, false → playAgain = false
💡 Підказка: Відкрий консоль браузера (F12 → вкладка Console), щоб побачити історію спроб!

Як перевірити роботу?

  1. Натисни F12 (або ПКМ → "Перевірити" / "Inspect")
  2. Перейди на вкладку Console (Консоль)
  3. Натискай кнопки та спостерігай за виводом у консолі
