Моє хобі

Моїм основним захопленням є програмування. Мені подобається процес створення чогось нового з чистого аркуша, від ідеї до готового продукту. Також я цікавлюсь фотографією та люблю проводити час на свіжому повітрі, граючи у футбол чи подорожуючи.

Програмування

#include <iostream>
using namespace std;

int main() {
    // Це приклад простої програми на C++
    int number;

    cout << "Введіть число: ";
    cin >> number;

    if (number % 2 == 0) {
        cout << number << " - парне число." << endl;
    } else {
        cout << number << " - непарне число." << endl;
    }

    return 0;
}
